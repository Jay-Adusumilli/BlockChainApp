This folder is for holding the setup scripts for the GENI nodes and server.
If a script needs to be updated, make sure to not change its name or path in the repository.
If the name or path is changed, all the rspec files will need to be changed.



Server:
URL: https://github.com/Jay-Adusumilli/BlockChainApp/blob/main/setup-scripts/server-setup.tar.gz
Install Path:
/local
Command:
sh /local/server-setup.sh

Node: 
URL: https://github.com/Jay-Adusumilli/BlockChainApp/blob/main/setup-scripts/node-setup.tar.gz
Install Path:
/local
Command:
sh /local/node-setup.sh
